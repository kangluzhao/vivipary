# Description
1. Interspecies orthologous relationships, generated by OrthoFinder v3, are provided in Orthogroups.GeneCount.tsv and Orthogroups.rar, serving as primary source files for the analyses in this study.
2. Code.R contains the main code used in this study, serving as a supplement to the Materials and Methods.
